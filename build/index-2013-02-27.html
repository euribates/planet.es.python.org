Genbeta:dev: Se hace saber que Django 1.5 ha llegado

Expect me nigga like you'd expect jesus to come back, expect me nigga, I'm coming

Obviamente, por la temática de este blog, el Django 1.5 que ha llegado no es una nueva versión del director de 'Django Desencadenado', la oscarizada y vengativa peli del oeste de Quentin Tarantino, sino que se trata de una nueva versión del popular framework para desarrollo en Python que desde ayer mismo está disponible.

Este nuevo Django 1.5 viene cargadito de novedades por lo que es recomendable darle un vistazo amplio a las release notes. De todos modos, así a groso modo, destaca sobre todo lo siguiente:

  • Modelo de usuario configurable
  • Soporte para salvar un subconjunto de los campos de un modelo
  • Cacheo de instancias relacionadas de un modelo
  • Soporte para streaming
  • Una nueva etiqueta de bloque llamada verbatim
  • Soporte para Python 3 (todavía experimental)

Además, por supuesto, un buen puñado de bugs arreglados y mejoras ostensibles en la documentación. En definitiva, que si eres djanguero (¿o será djanguista?) ya estás tardando en probar esta nueva versión y compartir tus impresiones con el resto de genbetadeveros.

Pd: la cita del principio es un verso del tema 'Untouchable' del fenecido 2Pac que se usa en 'Django Desencadenado'
